# Genomic_variant-analysis
Es una prueba para aprender a usarlo



_Título_	                      # Análisis de Variantes Genómicas (Genomic Variant Analysis)
_Descripción_	                  Proyecto simple de programación científica para el análisis de archivos de variantes genéticas (VCF). Desarrollado como parte de la asignatura de                                       Programación Científica del Máster en Bioinformática.
_Instalación_
1. Clonar el repositorio: git clone https://github.com/tu_usuario/Genomic_Variant_Analysis.git


2. Instalar dependencias: pip install -r requirements.txt

_Uso_	
Ejecutar el script principal: python scripts/variant_analyzer.py


El script procesará el archivo de ejemplo en data/sample.vcf.
